# Linux and Docker Workshop (soar-3dc)

A hands-on introduction to Linux, the terminal, dotfiles and Docker, run as a collaboration between SOAR and 3dc.

## Get started (before or at the workshop)

You need only a GitHub account and a browser. No installs.

1. Open this repo.
2. Click the green Code button.
3. Pick the Codespaces tab.
4. Click "Create codespace on main".
5. Wait 2 to 4 minutes while it sets up.
6. Open CHEATSHEET.md and run section 1.

## What is in this repo

- `.devcontainer/` defines the codespace itself, the machine you are working in.
- `dotfiles/` a small set of config files you apply to your own shell.
- `myapp/` a tiny web app you will build and run with Docker.
- `CHEATSHEET.md` every command you need, in order.

## After the workshop (take-home)

Your codespace keeps existing at github.com/codespaces. Personal accounts get a free monthly quota. GitHub can also auto-apply a repo named `dotfiles` to every codespace you ever create, see [personalizing GitHub Codespaces for your account](https://docs.github.com/en/codespaces/setting-your-user-preferences/personalizing-github-codespaces-for-your-account#dotfiles).

To go further at home, install Docker Desktop, or try Ubuntu in a VM (UTM on Apple Silicon, VirtualBox on everything else).
